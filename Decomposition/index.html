Démonstration opérateur de décomposition

Pour voir un résultat veuillez regarder dans la console de votre navigateur.(f12)

Création de plusieur fonction utilisant l'opérateur de décomposition : '...' .
la fonction concat() peut recevoir de 0 à x mots, qui vont être concaténés pour former une phrase.
la fonction concatRepeat() doit recevoir un premier paramètre indiquant le nombre de répétition de la phrase finale, et peut recevoir de 0 à x mots, qui vont être concaténés pour former une phrase.
l'opérateur de décomposition ne peut être utilisé qu'une seule fois dans les paramètres d'une fonction, et doit toujours être le dernier paramètre.
